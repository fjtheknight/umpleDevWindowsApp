# umpleDevWindowsApp

This is an application written in Java to help you with Umple development on Windows.

Put this application folder in the same directory as your umple folder.

Run "app.bat" as administrator.

## Buttons

![umpleDevWindowsApp](https://i.imgur.com/PhsY2rE.png)

Four states were used for this project:

* Build Umple First: gets the Umple compiler used to compile Umple itself
* Build Umple: full build used to compile and test fully
* Build Umple Online: applies changes to your local version of UmpleOnline
* Build Umple Quick: does a quick full compile of your code without testing
* Build Doc: builds documentation
* Run Apache: runs an Apache service
